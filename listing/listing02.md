Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Вывод:
2
1

Ф-ии defer располагаются в стеке по принципу LIFO. Т.е. первым выполняется ф-ия, объявленная последней.

Вызов функции test(). В результате будет возвращено значение = 2, т.к. в данном случае ф-ия в defer является замыканием. Произошел захват переменной x (возвращаемое значение), поэтому результат сохранится в ту область стека, где хранится переменная x.

Вызов функции anotherTest(). В результате будет возвращено значение = 1. Замыкание также произойдет на переменной x (объявлена в ф-ии через var), но ее значение будет хранится в другой области стека (не в той же, где хранится значение переменной x внутри отложенной функции),
поэтому даже после того, как отложенная функция отработает, значение останется прежним = 1.
```
